BetterUnc Documentation

Get Started

Lets load the library then we can continue to the documentation

local BetterUnc = loadstring(game:HttpGet("https://raw.githubusercontent.com/zwar808/BetterUNC/main/main.lua"))()

Functions

GetExecutorVersion

Returns the version of the executor.

local version = BetterUnc.GetExecutorVersion()

GetLibraryVersion

Returns the version of the BetterUnc library.

local version = BetterUnc.GetLibraryVersion()

Get

Sends an HTTP GET request to the specified URL and returns the response.

local response = BetterUnc.Get("https://example.com")

Post

Sends an HTTP POST request with the given data to the specified URL.


local data = { key = "value" }
local response = BetterUnc.Post("https://example.com", data, "application/json")

Put

Sends an HTTP PUT request to update data at the specified URL.


local data = { key = "new value" }
local response = BetterUnc.Put("https://example.com", data, "application/json")

Delete

Sends an HTTP DELETE request to remove data from the specified URL.

local response = BetterUnc.Delete("https://example.com")

Patch

Sends an HTTP PATCH request to partially update data at the specified URL.


local data = { key = "updated value" }
local response = BetterUnc.Patch("https://example.com", data, "application/json")

Head

Sends an HTTP HEAD request to retrieve metadata from the specified URL.

local response = BetterUnc.Head("https://example.com")

Options

Sends an HTTP OPTIONS request to retrieve the allowed methods from the specified URL.

local response = BetterUnc.Options("https://example.com")

Trace

Sends an HTTP TRACE request to retrieve the request message as received by the server.

local response = BetterUnc.Trace("https://example.com")

WebSocketConnect

Establishes a WebSocket connection to the specified URL.

local connection = BetterUnc.WebSocketConnect("wss://example.com/socket")

WebSocketSend

Sends a message over the WebSocket connection.


local message = "Hello, World!"
local success = BetterUnc.WebSocketSend(connection, message)

WebSocketReceive

Receives a message from the WebSocket connection.

local message = BetterUnc.WebSocketReceive(connection)

WebSocketClose

Closes the WebSocket connection.

local success = BetterUnc.WebSocketClose(connection)

GetPlayerByName

Finds and returns a player by their name.

local player = BetterUnc.GetPlayerByName("PlayerName")

GetAllPlayers

Returns a list of all players in the game.

local players = BetterUnc.GetAllPlayers()

KickPlayer

Kicks a player from the game with a given reason.


local player = BetterUnc.GetPlayerByName("PlayerName")
local success = BetterUnc.KickPlayer(player, "Reason for kicking")

ChangePlayerTeam

Changes the team of a player to the specified team name.


local player = BetterUnc.GetPlayerByName("PlayerName")
BetterUnc.ChangePlayerTeam(player, "TeamName")

SetPlayerName

Sets a new name for the player.


local player = BetterUnc.GetPlayerByName("PlayerName")
local success = BetterUnc.SetPlayerName(player, "NewName")

GetPlayerData

Retrieves basic data about a player.

local data = BetterUnc.GetPlayerData(player)

SetPlayerWalkSpeed

Sets the walk speed of a player’s character.


local player = BetterUnc.GetPlayerByName("PlayerName")
BetterUnc.SetPlayerWalkSpeed(player, 16)

SetPlayerJumpPower

Sets the jump power of a player’s character.


local player = BetterUnc.GetPlayerByName("PlayerName")
BetterUnc.SetPlayerJumpPower(player, 50)

SetGlobalData

Sets global data in the DataStore.

BetterUnc.SetGlobalData("key", "value")

GetGlobalData

Retrieves global data from the DataStore.

local value = BetterUnc.GetGlobalData("key")

TeleportPlayer

Teleports a player to a specific position.


local player = BetterUnc.GetPlayerByName("PlayerName")
BetterUnc.TeleportPlayer(player, Vector3.new(0, 50, 0))

PlaySound

Plays a sound at a specific position in the workspace.


BetterUnc.PlaySound("rbxassetid://123456789", Vector3.new(0, 50, 0))

CreatePart

Creates a new part with the specified size, color, and position.


local part = BetterUnc.CreatePart(Vector3.new(4, 1, 2), Color3.fromRGB(255, 0, 0), Vector3.new(0, 10, 0))

ChangePartColor

Changes the color of a part.


local part = workspace:FindFirstChild("PartName")
BetterUnc.ChangePartColor(part, Color3.fromRGB(0, 255, 0))

RemovePart

Removes a part from the workspace.


local part = workspace:FindFirstChild("PartName")
BetterUnc.RemovePart(part)
